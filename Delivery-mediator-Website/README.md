Flowchart Description:
The flowchart presents the end-to-end delivery workflow followed by the Smart Delivery Mediator system. It begins with order collection from multiple e-commerce platforms, followed by package validation based on size, weight, and destination. The system then selects the nearest warehouse, assigns the appropriate transportation mode, and dispatches the shipment with invoice generation. The final stage enables live tracking until the package is successfully delivered. This workflow clearly explains how the platform coordinates multiple logistics steps in a structured and efficient sequence.

Prototype Description:
The prototype demonstrates a Smart Delivery Mediator dashboard designed to simulate logistics coordination and monitoring. It includes user login, order collection from different platforms, revenue tracking, warehouse allocation, price analysis, shipment tracking, and automated alert generation. The dashboard visualizes order flow stages such as registered, in-transit, and delivered, while also providing real-time updates and system insights. Overall, the prototype showcases how a centralized interface can streamline order management, logistics planning, and delivery monitoring in a single platform.
