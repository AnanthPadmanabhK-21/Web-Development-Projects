Backlog Intelligence Dashboard-Description:
BacklogPro is a frontend prototype of an enterprise backlog intelligence and task management dashboard. The system simulates role-based access where Admin users can create projects, add tasks, assign owners, and run an automated cleanup engine to detect duplicates, prioritize tasks, and update task statuses. The dashboard provides real-time analytics using KPI cards and charts, along with PDF report generation for backlog insights. This prototype demonstrates how organizations can monitor project backlogs, automate task classification, and generate analytics-driven reports.

Food Delivery Web App-Description:
FoodDash is a frontend prototype of a modern food delivery web application that simulates the complete ordering workflow. The system includes login, restaurant browsing, smart recommendations, cart management with GST and delivery charges, address and payment flow, order history tracking, and delivery tracking simulation. It also features an admin panel to add new outlets and a search system for restaurants. This prototype demonstrates the end-to-end user experience of an online food ordering and delivery platform
