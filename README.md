# Web-Development-Projects
This repository contains my web development practice projects created using HTML, CSS and basic UI/UX layout principles. It includes multiple mini web pages and prototypes such as a backend collision management flowchart, a food ordering website prototype, a backlog/task management prototype, and additional webpage experiments built to practice page structuring, layout design, and real-world application concepts. Overall, this repository represents my learning journey in building simple static websites and improving my front-end fundamentals.
